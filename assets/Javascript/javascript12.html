    • 01. 데이터 이펙트 : 1부터 100까지 출력하기

    for문을 이용해서 0-100까지 출력하는 예제입니다.

     for(let i=1; i<=100; i++) { // 반복문 주석 표시
        console.log(i); // 초기값을 0으로 설정하고 1부터 100까지 출력함.
        }

// 1,2,3,4,5,6~~100

    02. 데이터 이펙트 : 0부터 50까지 출력하기

    for문을 이용해서 1-50까지 출력하는 예제입니다.

     for(let i=1; i<=50; i++) { // 반복문 주석 표시
            console.log(i); // 초기값을 0으로 설정하고 1씩 증가시켜서 50번 반복함.
        }
    
// 1,2,3,4,5,6~~50
