01. if문

조건문(conditional statement)의 하나로, 주어진 조건을 평가한 후, 그 결과에 따라 프로그램의 실행 흐름을 결정합니다.

{
    // false : 0, null, undefined, false, ""(빈문자열)
    // true : 1, 2, "0", "1", "ABC", [], {}, true
    if(조건식){
        document.write("실행되었습니다.(true)");
    } else {
        document.write("실행되었습니다.(false)");
    }
}
결과 확인하기
실행되었습니다.(false)

02. if 문 - 생략가능

조건식이 간단한 경우에 중괄호({})를 생략하여 간단하게 표현하는 것을 말합니다.

{
    const num = 100;                                                    
        if(num) document.write("실행되었습니다.(ture)");
        else document.write("실행되었습니다.(false)"); //(이렇게 생략가능) 
}
결과 확인하기
실행되었습니다.(ture)

04. 다중 if 문

조건에 따라 여러 개의 명령문 중 하나를 실행하도록 하는 조건문입니다.

이는 프로그래밍 언어에서 가장 기본적인 제어 구조 중 하나입니다.

{
    const num = 90;

if (num == 90) {
    document.write("실행되었습니다(num == 90)");
} else if(num == 100){
    document.write("실행되었습니다(num == 100)");
    
} else if(num == 110){
    document.write("실행되었습니다(num == 110)");
    
} else if(num == 120){
    document.write("실행되었습니다(num == 120)");

} else {
    document.write("실행되었습니다");
    
}
결과 확인하기
실행되었습니다(num == 90)

05. 중첩 if문

if문 안에 또 다른 if문을 넣어 여러 조건을 체크하는 제어 구조입니다.

즉, 조건문 내부에 다른 조건문이 들어가는 구조를 말합니다.

{
    const  num = 100;
                                    
    if(num == 100){
        document.write("실행되었습니다.(1)");
        if(num == 100) {
            document.write("실행되었습니다.(2)");
                if(num == 100){
                document.write("실행되었습니다.(3)")
            }   
        }
    } else {
        document.write("실행되었습니다.(4)")
    }
}
결과 확인하기
실행되었습니다.(1)실행되었습니다.(2)실행되었습니다.(3)

06. switch 문

조건문의 일종으로, 주어진 표현식(expression)의 값에 따라서 실행할 코드 블록을 선택하는 구문입니다.

{
    const num  = 100;

    switch(num){
        case 90:
            document.write("실행90");
            break;
        case 80:
            document.write("실행80");
            break;
        case 70:
            document.write("실행70"); 
            break;
        case 60:
            document.write("실행60");
            break; 
        case 50:
            document.write("실행50"); 
            break;
        default:
            document.write("0");    
    }
}
결과 확인하기
0

07. while 문

JavaScript에서 가장 기본적인 반복문 중 하나로, 조건식이 참(true)인 동안 반복해서 코드 블록을 실행합니다.

{
    for(let i=0; i<5; i++){     
        document.write(i);     
    }

    let num = 0;                
    while(num<5){               
        document.write(num);    
        num++;                        
    }
}
결과 확인하기

08. do while 문

while 문과 비슷한 반복문입니다.

do while 문은 코드 블록을 실행한 다음, 조건식이 참인 동안 반복해서 실행합니다.

{
    let num2 = 0;
    do {
        document.write(num2);       // 조건에 맞지 않아도 맞는게 있으면 한번은 실행됨 (죽은 문법)
        num2++;
        } while (num2<5);
    }
}

10. 중첩 for 문

중첩 for문은 첫번째 for문이 한번 실행되면 두번째 for문이 반복을 모두 끝난 후에 다시 첫번째 for문의 두번째가 실행됩니다. 

{
    for(let i=1; i<=10; i++){
        document.write(i,"
");
        for(let j=1; j<=10; j++){
            document.write(j,"
");
        }
    }
}

12.continue 문

현재 도는 레이블이 지정된 루프의 현재 반복에서 명령문의 실행을 종료하고 반복문의 처음으로 돌아가여 루프문의 다음 코드르 실행합니다.

{
    for(let i = 1; <20; i++){
        if(i == 10){
            continue;
        }
        document.write(i);

}
