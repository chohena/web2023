선언적 함수

함수 선언식(function declaration)으로 정의된 함수를 가리킵니다. 

{
    function func (){
        document.write("실행됐습니다.");
    }
    func ();
}
추가 설명
함수 선언식은 function 키워드를 사용하여 함수 이름, 매개 변수 및 함수 본문을 정의하는 방식입니다.
선언적 함수는 함수가 호출되기 전에 자바스크립트 엔진에 의해 처리됩니다. 따라서, 함수 호출 코드가 함수 정의보다 먼저 나오더라도 실행됩니다. 이는 함수 호이스팅(function hoisting)이라고도 알려져 있습니다.
선언적 함수는 간단하게 정의하고, 코드를 읽기 쉽게 만들며, 중복 코드를 방지하고 코드의 재사용성을 높이는 데 유용합니다.

익명 함수

함수 표현식(function expression)으로 정의된 함수를 가리킵니다.
함수 표현식은 함수 이름을 생략하고 변수에 함수를 할당하는 방식으로 정의하는 함수입니다. 

{
    const func = function(){
    document.write("실행되었습니다.!
");
    }
    func();
}
추가 설명
익명 함수는 함수 이름을 생략하므로, 변수에 할당된 함수를 호출하여 사용합니다.
익명 함수는 함수 선언식보다 유연하게 사용할 수 있습니다.
익명 함수는 코드 중복을 방지하고 가독성을 높이는 데 유용합니다.

매개변수 함수

함수가 호출될 때 함수 내부로 전달되는 값입니다.


함수는 이러한 매개변수를 이용하여 입력 값을 받아들이고, 처리 후 결과 값을 반환합니다. 
{
    function func(str){
    document.write(str);
    }
    func("실행되었습니다.!
");
    }
추가 설명
함수의 매개변수는 함수 내부에서 변수와 같은 역할을 합니다.
함수 내부에서 매개변수를 참조하여 값을 처리하거나 다른 변수에 할당할 수 있습니다.

리턴값 함수

함수가 처리를 마친 후 반환하는 값을 의미합니다.


함수는 매개변수를 받아 처리한 후, 결과 값을 반환하게 됩니다. 
{
    function func() {
        const str = "실행되었습니다.!";
        return str;
        }
    document.write(func());
}
추가 설명
함수를 호출하면, 반환된 결과 값을 다른 변수에 할당하거나, 다른 함수에서 인수로 사용할 수 있습니다.

선언적 함수 => 화살표

함수가 처리를 마친 후 반환하는 값을 의미합니다.


함수는 매개변수를 받아 처리한 후, 결과 값을 반환하게 됩니다. 
{
    func() = () => {
    document.write("실행되었습니다.");
    }
    func();
    }
추가 설명
함수를 호출하면, 반환된 결과 값을 다른 변수에 할당하거나, 다른 함수에서 인수로 사용할 수 있습니다.
