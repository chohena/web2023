09. 배열 : 데이터 불러오기_for of()문

배열, 문자열, 맵, 셋 등의 객체에서 사용할 수 있는 반복문입니다.
이 반복문은 해당 객체의 각 요소를 하나씩 가져와서 반복 작업을 수행합니다.

{
    const arr = [100,200,300,400,500];

    for(let i of arr){
        document.write(i);
    }
}
결과 확인하기
100,200,300,400,500

10. 배열 : 데이터 불러오기_for in()문

객체의 속성(property)을 반복하면서 처리하는 자바스크립트의 반복문 중 하나입니다. 이 반복문은 객체의 속성들을 열거(enumeration)하기 위해 만들어졌습니다.
객체를 위한 배열문이기에 결과값을 i만 출력하게 되면 인덱스 값이 나옵니다. 하여 arr[i]로 결과값을 불러와야 합니다.

{
    const arr = [100, 200, 300, 400, 500];

    for(let i in arr){
        document.write(arr[i]);
    }
}
결과 확인하기
100,200,300,400,500

11. 배열 : 데이터 불러오기_ㅡmap()문

메서드는 배열의 각 요소를 변환하여 새로운 배열을 생성하는 메서드입니다. map() 메서드는 기존 배열을 변경하지 않고 새로운 배열을 반환합니다.
map() 메서드는 콜백 함수를 인수로 받습니다. 콜백 함수는 세 개의 인수를 받습니다. 첫 번째 인수는 현재 처리 중인 배열 요소입니다.
두 번째 인수는 현재 요소의 인덱스이며, 세 번째 인수는 map() 메서드가 호출된 배열 자체입니다.
콜백 함수는 배열 요소를 변환하여 반환합니다. map() 메서드는 이러한 반환 값을 사용하여 새로운 배열을 생성합니다.

{
    const num = [100, 200, 300, 400, 500];

    num.forEach(function(el, i, a){
            console.log(el)
            console.log(i)
            console.log(a)
    });

    num.map(function(e, i, a){
        console.log(el)
        console.log(i)
        console.log(a)
    });
}
결과 확인하기
100 0 100,200,300,400,500 200 1 100,200,300,400,500 300 2 100,200,300,400,5500 400 3 100,200,300,400,500 500 4 100,200,300,400,500

19. 객체 : 데이터 불러오기 : hasOwnProperty()

데이터가 있는 지 확인할 때 사용하는 속성

{
    const obj = {
        a: 100,
        a: 200,
        a: "javascrip",
    }
    console.log(obj.hasOwnProperty("a"));   //true
    console.log(obj.hasOwnProperty("b"));   //true
    console.log(obj.hasOwnProperty("c"));   //true
    console.log(obj.hasOwnProperty("d"));   //false

    console.log("a" in obj);            // 약식으로 사용하는 방법, 결과값은 동일함
    console.log("b" in obj);
    console.log("c" in obj);
    console.log("d" in obj);
}
결과 확인하기
true
true
true
false
