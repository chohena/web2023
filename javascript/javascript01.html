01. 변수 : 데이터저장

변수는 데이터를 저장하는 저장소입니다.

{
    var x = 100;
    var y = 200;
    var z = "javascript";
    
    console.log(x);
}
    
    {
    var x = 100;            // 변수 X에 숫자 100을 저장함
    var y = 200;            // 변수 y에 숫자 200을 저장함
    var z = "javascript";   // 변수 z에 문자 "javascript"를 저장함
    
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript
변수(variable)란 값을 저장할 수 있는 메모리 공간을 의미합니다.
변수는 이름을 가지고 있으며, 이를 통해 해당 변수에 저장된 값에 접근할 수 있습니다.

02. 변수 : 데이터저장 + 데이터 변경

변수는 데이터를 저장할 수도 있지만 변경도 가능합니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 200;
    z = "react";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과확인하기
300
200
react

03. 변수 : 데이터저장 + 데이터 변경 + 데이터 추가

변수는 데이터를 저장하고 변경하고 추가할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;   // x = x + 300
    y += 400;
    z += "react"

    console.log(x);
    console.log(y);
    console.log(z);
}
결과확인하기
400
600
javascriptreact

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수는 데이터 저장은 가능하나 변경은 불가능합니다.

{
    const x = 100;
    const y = 200;
    const z = "javascript";
    
    x = 300;   //Assignment to constant variable
    y = 400;
    z = "react"
            
    console.log(x);
    console.log(y);
    console.log(z);
}
결과확인하기
300
400

05. 배열: 데이터 저장(여러개): 표현방법1

배열은 여러개의 데이터를 배열한 순서대로 값을 저장합니다.

{
    const arr=new Array();
    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과확인하기
100
200

16.객체 : 데이터 저장 (키와 값) : 표현방법8

객체 안에는 실행문이 들어갈 수 있다.

 {
    const obj = {
        a: 100,
        b: [200,300],
        c: "javascript",
        d: function() {      //함수는 = function
            console.log("javascript가 실행되었습니다."); }
        e: function() {
            console.log(obj.c+ "javascript가 실행되었습니다.")
        }
    }

    console.log(obj.a);
    console.log(obj.b[0]);
    console.log(obj.b[1]);
    console.log(obj.b[2]);  //2는 없기 때문에 출력하면 콘솔에 undefined가 나옴.
    console.log(obj.c);
    console.log(obj.d);     //(X) document.write가 나오면 안됨.
    obj.d();                    // 실행문에는 저장문 다음에 ()를 붙여야 한다.ㅣ
}
결과확인하기
100
200
javascript
